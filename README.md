 KASEYSET – Бронирование отелей

KASEYSET— это полнофункциональное Android-приложение для бронирования отелей с разделением ролей (клиент и администратор). Приложение предоставляет удобный интерфейс для поиска, фильтрации и бронирования отелей, управления избранным, просмотра истории бронирований и бонусной программы. Для администратора доступны расширенные возможности: управление пользователями, отелями, бронями, отзывами и статистика.

 Функциональные возможности
 Для клиента
- Регистрация и вход (с сохранением сессии через SharedPreferences).
- Поиск отелей по названию или городу.
- Фильтрация:
  -VIP-отели (доступно только для VIP-клиентов).
  - По максимальной цене.
- Просмотр детальной информации об отеле: описание, удобства, отзывы.
- Бронирование отеля с указанием дат и количества гостей.
- Избранное: добавление/удаление отелей в избранное.
- Профиль:
  - Просмотр и редактирование личных данных.
  - Бонусный счёт и история бонусных операций.
  - Список текущих бронирований с возможностью отмены (для активных) и редактирования (для ожидающих).
  - Оставление отзыва после завершённого проживания.
- Связь с администратором: отображение контактных данных администратора.

Для администратора
- Панель статистики:
  - Общая статистика (количество пользователей, отелей, броней, выручка).
  - Детальная статистика с разбивкой по ролям и статусам.
- Управление пользователями:
  - Просмотр списка всех пользователей.
  - Изменение роли (обычный, VIP, администратор).
  - Удаление пользователя.
  - Детальная информация о пользователе (история броней и отзывов).
- Управление отелями
  - Просмотр списка отелей с количеством броней.
  - Добавление нового отеля (название, город, цена, рейтинг, описание, количество номеров, удобства, VIP-статус).
  - Редактирование и удаление отеля.
- Управление бронированиями:
  - Просмотр всех бронирований с фильтрацией по статусу и поиском.
  - Изменение статуса брони: подтверждение, отмена.
  - Редактирование дат и количества гостей.
  - Удаление брони.
  - Автоматический перевод завершённых броней в статус «Завершена» (по дате выезда).
- Управление отзывами:
  - Просмотр всех отзывов.
  - Удаление неподобающих отзывов.

Архитектура и структура проекта

 Архитектура
Приложение построено на основе **монолитной архитектуры** с явным разделением ответственности:
-Activity** (`MainActivity`) — единственная точка входа, управляет отображением экранов через динамическое добавление View. Используется ScrollView и LinearLayout для построения интерфейса.
- DatabaseHelper — класс-помощник для работы с SQLite, реализующий все операции с базой данных (CRUD для пользователей, отелей, бронирований, избранного, бонусов, отзывов). Содержит начальную инициализацию с тестовыми данными.
-Data classes — описаны внутри `DatabaseHelper.kt` для удобства.

Технологический стек
-Язык: Kotlin
- UI: XML-разметка, View-система (не Compose)
- База данных SQLite (через `SQLiteOpenHelper`)
- Библиотеки:
  - `androidx.appcompat` — для совместимости с более старыми версиями Android.
  - `com.google.android.material` — для использования Material Components (кнопки, карточки, диалоги).
  - `androidx.cardview` — для создания карточек отелей и бронирований.
  - `androidx.recyclerview` — для эффективного отображения списков.

Ключевые особенности реализации
- Динамическое построение интерфейса — все экраны создаются программно, без использования отдельных layout-файлов для каждого раздела. Это упрощает навигацию и переключение между ролями, но усложняет поддержку.
- Ролевая модель— после входа определяется роль пользователя (admin, vip, user), и в зависимости от роли показываются соответствующие вкладки и функционал.
- Работа с БД — все методы доступа к данным сосредоточены в `DatabaseHelper`, что обеспечивает централизованное управление и упрощает тестирование.
- Обратная связь  — для информирования пользователя об успешности действий используются тосты (`Toast`).
- Диалоги подтверждения — перед удалением или отменой выводится диалог с вопросом.

Установка и запуск
 Требования
- Android Studio (рекомендуется версия Hedgehog или более новая)
- JDK 11 или выше
- Android SDK с API 24+
Инструкция
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/kaseyset.git
2. Откройте проект в Android Studio.
3. Дождитесь синхронизации Gradle.
4. Подключите устройство или эмулятор (Android 7.0+).
5. Нажмите Run ▶.
 Тестовые аккаунты
- **Администратор**: `admin@hotel.com` / `admin123`
- **VIP-клиент**: `vip@hotel.com` / `vip123`
- **Обычный клиент**: `user@hotel.com` / `user123`
